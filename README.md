# Excel Project Portfolio

## Overview

This repository contains all Excel based projects that I have worked on focused on data analysis, reporting, and dashboard development.

Across these projects, the following tools and features are used:

* **Power Query** for data cleaning and transformation
* **Pivot Tables & Pivot Charts** for aggregation and analysis
* **Slicers** for interactive filtering
* **Excel formulas** for calculations and data manipulation
* **Dashboard design** for clear and structured data presentation
