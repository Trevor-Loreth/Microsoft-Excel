# Coffee Sales Dashboard (Excel + Python)

## Overview
This project explores a coffee sales dataset from raw data to an interactive Excel dashboard. The focus was to design an interactive Excel dashboard that provides clear, decision-focused insights into sales performance, customer behavior, and time-based trends. Python was first used for exploratory analysis to understand the structure of the data and identify useful directions for analysis. Those findings then guided how I handled the data within Power Query. Pivot charts and slicers were then used to build an interactive dashboard.


## Objectives

* Analyze coffee sales data to identify trends and patterns
* Create meaningful time-based groupings (time buckets)
* Build an interactive dashboard for non-technical users


## Features
**Microsoft Excel**
  * Power Query was used for correcting data types and creating time buckets
  * Pivot Tables & Pivot Charts (aggregation & visualization)
  * Slicers (interactivity)

**Python**
  * Exploratory Data Analysis (EDA)
  * Data profilng to understand the data structure and distribution
  * Identify trends and anomolies
  * Explore potential metrics


## Files Included

* `coffee_dashboard.xlsx` → Final dashboard (includes Power Query + visuals)
* `raw_data.xlsx` (or sheet) → Original dataset
* `transformed_data` (within Excel / Power Query) → Cleaned and structured data
* (Optional) Python scripts / notebook used for EDA
